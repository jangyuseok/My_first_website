Library & Framework란 무엇인가?

소프트웨어를 만드려면 혼자 만들 수 없다 WHY? 누군가 만들어 놓은 소프트웨어 안에서 내가 또다른 소프트웨어를 조각해 나가는거기때문에
=> Library & Framework : 다른 사람이 만들어 놓은것을 빌려 정리하는것

Library : Library는 도서관이라는 뜻 다른 사람들이 만들어 놓은 소프트웨어를 잘 정리 정돈한것
=> 내가 만들어 놓은 부품을 가져오는 기능 / 소프트웨어를 만드는사람이 라이브러리를 끌어와서 쓰는것

Framework : 우리가 만들고자 하는 것이 있을때 무엇을 만들고자에 따라 갈려지는 특성이 있을것
=> 채팅을 만들고자 할때 공통적인 분모가 있을것인데 Framework가 만들어 놓은것을
우리가 만들고자하는것의 특성을 살짝 살짝 변경하는것 / 프레임워크안에 우리가 들어가서 작업하는것
